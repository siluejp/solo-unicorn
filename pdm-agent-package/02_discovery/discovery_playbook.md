# Discovery Playbook（1週間で最大リスクを潰す）

## 目的
- 今週の「最大リスク」を1つ選び、検証して意思決定する

## 週の流れ（例）
- Mon: 仮説選定（HB上位）→ 実験カード起草
- Tue: プロト/LP/台本準備、計測設計確認
- Wed: 実施（インタビュー/テスト/小実験）
- Thu: 中間レビュー（続行/中止/修正）
- Fri: 分析→ Decision → vPBに受け渡し or HBへ戻す

## リスク種別の目安
- Value: そもそも欲しいか
- Usability: 使えるか
- Feasibility: 作れるか
- Viability: 売れる/成り立つか

## 参加者
- PdM（意思決定）
- UX（体験）
- Eng（実現性）
- Analytics（計測）
