# 競合分析 & スコアリング（Competitive Landscape）

## 評価軸の定義（全アイデア共通）

| 軸 | 見るポイント |
|----|--------------|
| 課題適合度 | 顧客の中核課題にどれだけ直撃しているか |
| 価値の明確さ | 提供価値が一言で説明できるか |
| 導入容易性 | 導入・利用開始までの心理/実務コスト |
| 価格妥当性 | 顧客の支払い感覚との一致 |
| ワークフロー定着 | 日常業務・生活に組み込まれやすいか |
| データ優位性 | 使うほど差が広がる構造があるか |
| 切替コスト | 他サービスから乗り換えにくいか |
| 防御力（堀） | 大手・既存プレイヤーに真似されにくいか |

- 評価：1（弱い）〜5（非常に強い）
- 合計点は **意思決定用の目安**（絶対評価ではない）
